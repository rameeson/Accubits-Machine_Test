# Project Title
    Accubits-Machine_Test

## Description
   To implement a project using node js with express js to achieve below functionalities
   -create add user API
   -create newsletter API
   -read input data from csv file
   -integrate Mongo DB
   -queue for email jobs using Rabbit MQ
   -send email to user using Nodemailer
   -send logs to DB if success
   -add data to parking lot queue incase of failure

## Requirements  
   -Node js
   -express js
   -Mongo DB
   -RabbitMQ
   -nodemailer
   -body-parser
   -csv-parser

## Steps to run the project
   Import project to any IDE (Example VS Code/Atom)
   Run the command 'npm install' in the terminal of VS Code
   Run the command 'node app.js' or 'nodemon app.js' in the terminal of VS Code
   Give the URL http://localhost:3000/ in a browser
   Tap on "Enter User Details" to add new user
   Tap on "Newsletter" to send newsletter to user's Email ID
